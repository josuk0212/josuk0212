기술적인 성장뿐만 아니라, 함께 일하는 동료들에게도 좋은 영향을 주는 개발자가 되고자 합니다. </br>
유연한 사고와 적극적인 소통을 바탕으로, 다양한 사람들과 협업하며 더 나은 서비스를 만들어 가고 싶습니다.
